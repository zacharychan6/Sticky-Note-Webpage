# Sticky Notes Webpage

A simple, colorful **sticky notes** web application built with **Django**. Create, view, edit, and delete digital sticky notes with a fun, random vibrant colors.


# Features

- **Create** new sticky notes with title and content
- **View** all notes in a colorful grid layout
- **Edit** and **Delete** existing notes
- Random vibrant colors for each sticky note
- Responsive and modern UI with Bootstrap + custom CSS
- Clean, sticky-note aesthetic with hover effects and smooth scrolling

# Tech Stack

- **Backend**: Django (Python)
- **Frontend**: HTML, CSS, JavaScript, Bootstrap 5
- **Database**: SQLite (default)
- **Templates**: Django Template Language
